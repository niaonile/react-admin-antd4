# 本项目是react为技术栈的后台项目

使用antd4、craco（配置扩展库）、dva、less等技术.

## 已有组件

tabs选项卡模式（layout布局），可切换普通路由模式或是tabs模式；  
全局content弹窗组件，已处理tabs模式下的兼容。
